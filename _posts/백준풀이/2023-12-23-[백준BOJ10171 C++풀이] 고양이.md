### 문제
[https://www.acmicpc.net/problem/10171](https://www.acmicpc.net/problem/10171)

```
\    /\
 )  ( ')
(  /  )
\(__)|
```

***

### 풀이
- \ 출력 방법은 다음과 같다.
```
cout << "\\" << endl;
```

***

### 제출

```
#include <iostream>
using namespace std;

int main() {
	cout << "\\    /\\" << endl;
	cout << " )  ( ')" << endl;
	cout << "(  /  )" << endl;
	cout << " \\(__)|";
	return 0;
}
```
