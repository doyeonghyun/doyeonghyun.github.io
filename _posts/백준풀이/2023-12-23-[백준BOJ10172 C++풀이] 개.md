> 아래 예제와 같이 개를 출력하시오.
>
> [https://www.acmicpc.net/problem/10172](https://www.acmicpc.net/problem/10172)

```
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
```

***

### 풀이
- \ 출력 방법은 다음과 같다.
```
cout << "\\" << endl;
```
- " 출력 방법은 다음과 같다.
```
cout <<"\"" << endl;
```
띄어쓰기 등의 사소한 문제로도 틀렸다고 나올 수 있으니 꼼꼼히 확인하는게 좋다.

***

### 제출
```C++17
#include <iostream>
using namespace std;

int main() {
	cout << "|\\_/|" << endl;
	cout << "|q p|   /}" << endl;
	cout << "( 0 )\"\"\"\\" << endl;
	cout << "|\"^\"`    |" << endl;
	cout << "||_/=\\\\__|";
	return 0;
}
```
